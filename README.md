# Investor Match MVP

An interactive prototype showing how founders can find matching impact investors.

## Quick Start

### View Live (GitHub Pages)
**Coming soon** — Enable Pages in repo settings:

2. Open in any browser

## For RevHub Demo

This prototype demonstrates:
- Founder input form (location, stage, industry, product)
- Investor type tags
- Check size slider ($50K → $10M+)
- Matched investors ranked by score

**Demo flow:**
1. Fill in venture details
2. Tap "Find My Investors"
3. See ranked matches with scores
